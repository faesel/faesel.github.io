5:T3ff2,<p>As a windows user the terminal experience has always been lacking, up till the new <a href="https://www.microsoft.com/en-gb/p/windows-terminal/9n0dx20hk701?rtc=1&activetab=pivot:overviewtab">windows terminal</a> was released. Incorporating <a href="https://docs.microsoft.com/en-us/windows/wsl/install-win10">WSL (Windows Subsystem for Lynx)</a> really helped bridge that gap as it opens up console experience that makes use of <strong>apt-get</strong> use the plethora of packages available. </p>
<p>Having tried using this for react apps I found the experience really slow when building apps. Running the same app in Powershell would start up in a fraction of the time. This got me thinking..</p>
<blockquote>
<p>Can I evolve my Powershell console experience in the same way I can with WSL?</p>
</blockquote>
<p>Looking at <a href="https://www.powershellgallery.com/">Powershells Gallery</a>, they have a total of <strong>7,091</strong> unique packages. There must be some things here we can use.</p>
<p>This article is about my journey from reading this blog post <a href="https://www.hanselman.com/blog/HowToMakeAPrettyPromptInWindowsTerminalWithPowerlineNerdFontsCascadiaCodeWSLAndOhmyposh.aspx">&quot;How to make a pretty prompt in Windows Terminal with Powerline, Nerd Fonts, Cascadia Code, WSL, and oh-my-posh&quot;</a> by Scott Hanselman, to taking the experience one step further to evolve my terminal. Video on it here:</p>
<p><a href="https://www.youtube.com/watch?v=lu__oGZVT98">https://www.youtube.com/watch?v=lu__oGZVT98</a></p>
<h2>Prerequisites</h2>
<p>This article assumes you have already read Hanselman&#39;s article and kitted your terminal out with the following (if not do so):</p>
<ul>
<li>Added Cascadia Ligature Font, for more compact and expressive text.</li>
<li>Added oh-my-posh, for better git branch information.</li>
</ul>
<h2>Making use of your Powershell profile</h2>
<p>Before your Powershell window starts, any code in your Powershell profile is executed first. You can find your profile directory location by typing the following in your console,</p>
<pre><code class="hljs "><span class="code-line"><span class="line-number">1</span><span class="line-content"><span class="hljs-meta"><span class="hljs-keyword">$PROFILE</span></span></span></span></code></pre><p>It&#39;s from this file we can begin importing in scripts and adding custom functions. If you followed the prerequisites, your profile file will be mostly empty except perhaps a command to set the theme in oh-my-posh.</p>
<pre><code class="hljs "><span class="code-line"><span class="line-number">1</span><span class="line-content"><span class="hljs-keyword">Set</span>-Theme <span class="hljs-comment">Paradox</span></span></span></code></pre><h2>Incorporating packages</h2>
<p>Before we start importing packages we need to get <strong>PowerShellGet</strong> setup, installation steps can be found here <a href="https://docs.microsoft.com/en-gb/powershell/scripting/gallery/installing-psget?view=powershell-7">Installing PowerShellGet</a>. Once complete you can start installing scripts like so,</p>
<pre><code class="hljs "><span class="code-line"><span class="line-number">1</span><span class="line-content">Install-<span class="hljs-keyword">Module</span> -<span class="hljs-keyword">Name</span> WifiTools </span></span></code></pre><p>That&#39;s it, you can actually start checking your wifi signal with <code>Show-WifiState</code> üì∂! Aside from wifi tools here are some choice selections, some of these are recommended as it helps create building blocks for your own custom scripts:</p>
<h3><a href="https://www.powershellgallery.com/packages/Terminal-Icons/0.1.1">1. Terminal Icons</a></h3>
<p><a href="https://www.powershellgallery.com/packages/Terminal-Icons/0.1.1">Terminal-Icons</a> really helps improve visibility when navigating. It also allows you to format your directory list wide so that you can see all the files and folders without scrolling down.</p>
<p>You can activate it in your Powershell profile by adding the following import</p>
<pre><code class="hljs "><span class="code-line"><span class="line-number">1</span><span class="line-content"><span class="hljs-keyword">Import</span>-<span class="hljs-keyword">Module</span> -<span class="hljs-keyword">Name</span> Terminal-Icons</span></span></code></pre><p>And that&#39;s it! witness the beauty:</p>
